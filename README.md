# School Management System

A modern School Management System built with:

- Next.js
- React
- TypeScript
- Tailwind CSS

## Project Goals

This project aims to provide schools with a complete management solution including:

- Student Management
- Class Management
- Subject Management
- Results Management
- Report Cards
- Attendance
- Teacher Management
- Fee Management (future)
- Parent Portal (future)

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
